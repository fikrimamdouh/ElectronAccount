# تطبيق المحاسبة - دليل التصميم الشامل

## منهجية التصميم

**النهج المختار**: Material Design System via MUI
**المبرر**: تطبيق محاسبة يركز على كثافة البيانات، الإنتاجية، والاستقرار. Material Design يوفر مكونات جاهزة، أنماط واضحة، ودعم ممتاز للغة العربية وRTL.

## المبادئ الأساسية

1. **الوضوح والقابلية للقراءة**: أولوية قصوى للبيانات المالية
2. **الكفاءة**: تقليل النقرات والخطوات للعمليات المتكررة
3. **الموثوقية**: تصميم مستقر وواضح يبني الثقة
4. **دعم RTL كامل**: جميع العناصر مُحسّنة للعربية

## Typography - الطباعة

**خطوط من Google Fonts**:
- Primary: 'Cairo' - خط عربي حديث واضح للواجهة
- Secondary: 'Tajawal' - للعناوين والنصوص الكبيرة

**التسلسل الهرمي**:
- العناوين الرئيسية: font-size 32-40px, font-weight 700
- عناوين الأقسام: font-size 24-28px, font-weight 600
- عناوين البطاقات: font-size 18-20px, font-weight 600
- النصوص الأساسية: font-size 16px, font-weight 400
- النصوص الثانوية: font-size 14px, font-weight 400
- الأرقام المالية: font-weight 600 للتمييز

## Layout System - نظام التخطيط

**Spacing Units** (Tailwind): استخدام 2, 4, 6, 8, 12, 16 كوحدات أساسية
- المسافات الصغيرة (داخل المكونات): p-2, p-4
- المسافات المتوسطة (بين العناصر): m-4, m-6, p-6
- المسافات الكبيرة (بين الأقسام): p-8, p-12, m-8

**Grid Structure**:
- Sidebar Navigation: 240px عرض ثابت
- Main Content Area: مرن، max-width 1400px
- Two-column layouts: 60-40 أو 50-50 حسب المحتوى

## Component Library - مكتبة المكونات

### Navigation

**Top AppBar**:
- ارتفاع 64px
- يحتوي: شعار التطبيق، عنوان الصفحة الحالية، أيقونة إشعارات، قائمة المستخدم
- Elevation: 1

**Sidebar Navigation**:
- عرض 240px، يمكن طيه إلى 64px (أيقونات فقط)
- أقسام رئيسية: لوحة التحكم، الحسابات، القيود اليومية، التقارير، الإعدادات
- استخدام MUI List و ListItem مع أيقونات Material Icons
- مؤشر بصري للصفحة النشطة

### Data Display Components

**Data Tables** (MUI Table):
- رؤوس أعمدة ثابتة عند التمرير
- تبادل ألوان الصفوف (zebra striping)
- إجراءات سريعة في كل صف (تعديل، حذف، عرض)
- pagination في الأسفل
- عرض 50-100 صف لكل صفحة

**Cards** (MUI Card):
- استخدام للإحصائيات في لوحة التحكم
- Elevation: 2
- padding: p-6
- تحتوي: أيقونة، عنوان، قيمة رقمية كبيرة، نسبة التغيير

**Forms**:
- استخدام MUI TextField مع variant="outlined"
- تسميات واضحة فوق الحقول
- رسائل خطأ تحت الحقل مباشرة
- تجميع الحقول ذات الصلة في أقسام
- spacing بين الحقول: gap-4 أو gap-6

### Specialized Components

**القيد المحاسبي**:
- جدول بصفين: مدين/دائن
- أزرار إضافة/حذف صف
- حساب تلقائي للمجاموع
- تنبيه بصري عند عدم توازن القيد

**ميزان المراجعة**:
- جدول بأعمدة: رقم الحساب، اسم الحساب، مدين، دائن، الرصيد
- صف إجمالي في الأسفل
- إمكانية التصفية والبحث

**Dashboard Cards**:
- 4 بطاقات إحصائية في صف: الإيرادات، المصروفات، الأرباح، الرصيد
- استخدام grid: grid-cols-1 md:grid-cols-2 lg:grid-cols-4
- gap-6 بين البطاقات

### Interactive Elements

**Buttons** (MUI Button):
- Primary: variant="contained" للإجراءات الرئيسية
- Secondary: variant="outlined" للإجراءات الثانوية
- Text buttons: للإجراءات الأقل أهمية
- أحجام: small, medium, large حسب السياق
- استخدام startIcon و endIcon حسب الحاجة

**Dialogs/Modals**:
- MUI Dialog للنماذج والتأكيدات
- max-width: sm للتأكيدات، md للنماذج المتوسطة، lg للنماذج المعقدة
- Actions في أسفل يمين (يسار في RTL)

**Snackbars**: 
- للإشعارات والتأكيدات
- موضع: bottom-left (bottom-right في RTL)
- مدة: 4-6 ثوانٍ

## Page-Specific Layouts

### لوحة التحكم
- 4 بطاقات إحصائية في الأعلى
- رسوم بيانية (2 columns): الإيرادات vs المصروفات الشهرية، توزيع المصروفات
- جدول آخر المعاملات (10 صفوف)

### صفحة القيود
- فلاتر في الأعلى (تاريخ، نوع القيد، الحالة)
- زر "قيد جديد" بارز
- جدول القيود مع pagination
- Dialog للإضافة/التعديل

### صفحة التقارير
- قائمة جانبية للتقارير المتاحة
- منطقة عرض التقرير مع خيارات طباعة/تصدير
- فلاتر للفترة الزمنية

## Accessibility & RTL

- جميع النصوص والعناصر محاذاة RTL
- استخدام dir="rtl" على المستوى العلوي
- أيقونات الأسهم معكوسة حسب الاتجاه
- دعم keyboard navigation كامل
- ARIA labels بالعربية

## Animation Guidelines

**تقليل الحركة**: استخدام transitions خفيفة فقط
- Page transitions: fade-in بسيط 200ms
- Dialog open/close: MUI defaults
- Button hover: تغيير subtle في opacity
- لا animations معقدة للبيانات المالية
