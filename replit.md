# RinaPro Business - نظام المحاسبة الاحترافي

## نظرة عامة
تطبيق ويب متكامل لإدارة الحسابات المحاسبية بواجهة عربية كاملة مع دعم RTL (الكتابة من اليمين إلى اليسار). يستخدم React و Material-UI لواجهة احترافية وحديثة.

## المميزات الرئيسية
- ✅ واجهة مستخدم عربية كاملة مع دعم RTL
- ✅ نظام محاسبة بالقيد المزدوج
- ✅ لوحة تحكم تفاعلية مع إحصائيات مالية
- ✅ إدارة شاملة للحسابات المالية
- ✅ تسجيل القيود المحاسبية اليومية
- ✅ ميزان المراجعة
- ✅ التقارير المالية (قائمة الدخل والميزانية العمومية)
- ✅ نظام البحث والتصفية
- ✅ دعم الوضع النهاري والليلي
- ✅ تصميم متجاوب يعمل على جميع الأجهزة

## البنية التقنية

### Frontend
- React 18 مع TypeScript
- Wouter للتوجيه
- TanStack Query لإدارة الحالة والبيانات
- Shadcn UI للمكونات
- Tailwind CSS للتصميم
- Recharts للرسوم البيانية
- خطوط Cairo و Tajawal العربية

### Backend
- Express.js
- In-memory storage (MemStorage)
- Zod للتحقق من البيانات
- TypeScript

## نماذج البيانات

### Accounts (الحسابات)
- id: معرف فريد
- code: رمز الحساب
- name: اسم الحساب
- type: نوع الحساب (أصول، خصوم، إيرادات، مصروفات، حقوق ملكية)
- balance: الرصيد
- isActive: حالة الحساب

### Entries (القيود)
- id: معرف فريد
- entryNumber: رقم القيد
- date: تاريخ القيد
- description: وصف القيد
- totalDebit: إجمالي المدين
- totalCredit: إجمالي الدائن
- isBalanced: هل القيد متوازن

### Entry Lines (سطور القيد)
- id: معرف فريد
- entryId: معرف القيد
- accountId: معرف الحساب
- debit: المبلغ المدين
- credit: المبلغ الدائن
- description: الوصف

## التصميم

### الألوان
- يستخدم نظام ألوان احترافي يناسب التطبيقات المحاسبية
- دعم كامل للوضع النهاري والليلي
- ألوان متباينة للقراءة الواضحة

### الخطوط
- Cairo: الخط الأساسي للواجهة
- Tajawal: للعناوين والنصوص الكبيرة
- خطوط أحادية المسافة للأرقام المالية

### المسافات
- نظام مسافات متسق باستخدام Tailwind
- مسافات صغيرة (p-2, p-4)
- مسافات متوسطة (p-6, m-6)
- مسافات كبيرة (p-8, p-12)

## الصفحات والمكونات

### لوحة التحكم (Dashboard)
- 4 بطاقات إحصائية: الإيرادات، المصروفات، الأرباح، الرصيد
- رسم بياني للإيرادات والمصروفات الشهرية
- قائمة بآخر القيود المحاسبية
- زر سريع لإضافة قيد جديد

### الحسابات (Accounts)
- عرض الحسابات مجمعة حسب النوع
- نموذج لإضافة حساب جديد
- بحث وتصفية
- عرض الرصيد والحالة لكل حساب

### القيود اليومية (Entries)
- قائمة بجميع القيود المحاسبية
- نموذج متقدم لإضافة قيد جديد
- دعم القيد المزدوج
- التحقق التلقائي من توازن القيد
- إمكانية إضافة سطور متعددة

### ميزان المراجعة (Trial Balance)
- عرض جميع الحسابات مع أرصدتها
- حساب تلقائي للمجاميع
- التحقق من توازن الميزان
- بحث وتصفية

### التقارير المالية (Reports)
- قائمة الدخل: عرض الإيرادات والمصروفات وصافي الربح
- الميزانية العمومية: عرض الأصول والخصوم وحقوق الملكية
- فلاتر للفترة الزمنية
- إمكانية التصدير

## واجهات API (سيتم تنفيذها في المرحلة التالية)

### Accounts
- GET /api/accounts - جلب جميع الحسابات
- POST /api/accounts - إضافة حساب جديد
- PUT /api/accounts/:id - تعديل حساب
- DELETE /api/accounts/:id - حذف حساب

### Entries
- GET /api/entries - جلب جميع القيود
- POST /api/entries - إضافة قيد جديد
- GET /api/entries/:id - جلب قيد محدد

### Reports
- GET /api/trial-balance - جلب ميزان المراجعة
- GET /api/reports/income-statement - جلب قائمة الدخل
- GET /api/reports/balance-sheet - جلب الميزانية العمومية

### Dashboard
- GET /api/dashboard/stats - جلب إحصائيات لوحة التحكم
